----------------------------------------

🚀 ШАГ 3: Запустите приложение
----------------------------------------

1. Сначала установите Node.js (если не установлен):
   https://nodejs.org/ - скачайте LTS версию

2. Перезапустите терминал после установки Node.js

3. В терминале выполните:
   cd C:\Users\1\Desktop\ddd\End-clone-dua
   npm install
   npm run dev

4. Откройте браузер: http://localhost:3000
